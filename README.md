# PokeVisor

Program do rozpoznawania obrazu z gry w pokera oraz wizualizacji stanu gry na komputerze.
Pisany jako projekt zaliczeniowy na Politechnice Poznańskiej.

## Autorzy
- Juliusz Horowski
- Michał Kalinowski
- Tomasz Kiljańczyk
- Szymon Wasilewski
